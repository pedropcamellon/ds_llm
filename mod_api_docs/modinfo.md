# modinfo.lua Reference

Source: https://dst-api-docs.fandom.com/wiki/Modinfo.lua

> This is a Lua script — you can use variables and functions to avoid repetition.

## Full Example

```lua
name = "Your mod name"

description =
[[
Shown in-game as a short description.
Width limit : 52 characters
Height limit: 17 rows
]]

author  = "Name yourself"
version = "1.0.0"
-- Versioning convention:
--   1.0.1 → 1.0.2  bug fix
--   1.0.1 → 1.1.0  new feature
--   1.0.1 → 2.0.0  breaking / save-incompatible change

forumthread = ""
-- Link to a Klei forum thread. Strip the base URL prefix; the game adds it.

api_version = 10   -- current API version for DST

-- Compatibility flags
dst_compatible              = true
dont_starve_compatible      = false
reign_of_giants_compatible  = false
hamlet_compatible           = false
forge_compatible            = false

all_clients_require_mod = true
-- false only for pure client-side UI mods that don't affect the world.

icon_atlas = "modicon.xml"
icon       = "modicon.tex"

server_filter_tags = { "something", "relevant", "to", "your", "mod" }

configuration_options = {
    -- see below
}
```

## configuration_options Fields

```lua
{
    name  = "someName",         -- key used in GetModConfigData("someName")
    label = "yourTitle",        -- shown to players (max ~27 chars on most screens)
    hover = "hint text",        -- tooltip on mouse-over (~156 chars, wraps to 2 rows)
    options = {
        { description = "Off", data = false },   -- description max 14 chars
        { description = "On",  data = true  },
    },
    default = false,
}
```

### Reading Options in Scripts

```lua
local value = GetModConfigData("someName")
```

### Making a Section Title (cosmetic separator)

```lua
{
    name    = "section_header",
    label   = "-- Section --",
    hover   = "",
    options = { { description = "", data = false } },
    default = false,
}
```

## Mod Icon

Requires two files generated by the Klei texture tool:

| File | Description |
|------|-------------|
| `modicon.tex` | Compiled texture |
| `modicon.xml` | Atlas descriptor |

Reference in `modinfo.lua`:
```lua
icon_atlas = "modicon.xml"
icon       = "modicon.tex"
```

## Text Icons

Special Unicode characters can be embedded in any `label`, `description`, or
`hover` string and will render as in-game icons. (Look up the DS character map
for the full list.)
